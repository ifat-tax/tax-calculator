חיבור ל-Google Drive — Service Account

הגדרה חד-פעמית — 4 צעדים:
1. ב-Google Cloud Console (פרויקט "ifat tax") → IAM & Admin → Service Accounts
2. לחץ "+ Create Service Account" → תן שם (למשל tax-sync) → Create and Continue → Done
3. לחץ על ה-Service Account שנוצר → טאב "Keys" → "Add Key" → "Create new key" → JSON → Create
   ↳ קובץ JSON יורד למחשב שלך
4. פתח את הקובץ בעורך טקסט → העתק את כל התוכן → הדבק למטה
✅ לא פג תוקף לעולם. מוגדר פעם אחת ועובד לצמיתות על כל מחשב.
